# 小程序简介
小程序使用原生微信小程序语言开发，后台采用springboot架构设计。小程序接入chatgpt，智能语音识别，图像识别等，微信支付等技术，包含智能AI聊天流式响应，语音识别，AI图像生成，会员管理、可以借助小程序生成 文章、古诗、论文、工作小结、日报月报、短视频脚本、各类语言程序源代码等
# 软件架构
###   前台：
- 微信小程序原生开发 
###   后台：
- java
- springboot
- mysql
- 百度、腾讯、科大讯飞语音识别技术
- 微信支付
